General Manual

Database

Database gathers file data from your system and displays them on screen

Functions:

Gather Data (Ctrl + G): Gather Data from the directory specified
Update Data (Ctrl + U): Re-gather data from the current directory
Import Data (Ctrl + I): Import data from an external .csv file
Plot Data (Ctrl + P): Plot (Histogram) the loaded data
Export As...: Export data as .csv/.txt/.xlsx/.json file
Sorting: Clicking on the respective data button sorts the data displayed

Searching:

Operators Info
=> > < <=key eg. <=20mb
RegEx
^key Starts with key
key$ Ends with key
%key% Case insensitive search
!key Not in name
basename: key Looks for matches only each basename
key && key Dual search
