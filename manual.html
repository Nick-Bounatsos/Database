General Manual

Database

Database gathers file data from your system and displays them on screen

Functions:

Gather Data (Ctrl + G): Gather Data from the directory specified
Update Data (Ctrl + U): Re-gathers data from the current directory
Import Data (Ctrl + I): Loads data from the .json file selected
Plot Data (Ctrl + P): Plots the data processed or searched
Export As...: Exports data as .csv/.txt/.xlsx file
Sorting: Clicking on the respective data button sorts the data displayed

Searching:

Operators
> < = {key} (eg. kb, bytes, GB etc)
RegEx (Name search):
^{key} → Starts with key
${key} → Ends with key
%{key}% → Caseless
!{key} → Not in name
basename: {key}
Dual search: &&
